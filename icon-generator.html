🌿 Wandermark Icon Generator

Click "Generate All Icons" to create every icon size needed for the Play Store and PWA.

📁 Where to put these files

1
Create a folder called icons/ in the same folder as your index.html
2
Save each downloaded icon into that folder with the exact filename shown below each preview
3
Your folder structure should be:
index.html
manifest.json
service-worker.js
icons/icon-192.png
icons/icon-512.png
... etc
4
Upload all files to GitHub Pages or your hosting provider
5
Go to pwabuilder.com and enter your site URL
